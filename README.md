## How to Run the Script
1. **Clone the Repository**:
    ```bash
    git clone git@github.com:BenafshaShakari/CIS2023-Linux-FinalProject.git
    cd CIS2023-Linux-FinalProject
    ```

2. **Make the Script Executable**:
    ```bash
    chmod +x print_numbers.sh
    ```

3. **Run the Script**:
    ```bash
    ./print_numbers.sh
    ```
